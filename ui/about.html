Block Overflow: incentivized, unbiased voting.

Take the idea of the traditional forum Q/A voting system and add Solidity contracts: this is Block Overflow.

Using Solidity smart contracts, the Block Overflow team has developed a secure, inventivized, token voting system using the Ethereum blockchain. With a token value of their choosing, users can stake an amount on their vote.
This promotes higher vote quality and helps keep out the trolls.
All users are provided with a base token amount, which will change based on the amounts they put down and the overall question votes.
