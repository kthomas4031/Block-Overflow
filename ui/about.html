ABOUT B L O C K O V E R C H A I N

Block Overchain: incentivized, unbiased voting.

Take the idea of the traditional forum Q/A voting system and add Solidity contracts: this is Block Overchain.

Using the Solidity language, the Block Overchain team has developed a secure, inventivized, token voting system using the blockchain. With a token value of their choosing, users can stake an amount on their vote.
This promotes higher vote quality and helps keep out the trolls.
All users are provided with a base token amount, which will change based on the amounts they put down and the overall question votes.
